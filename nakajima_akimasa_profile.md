## 2. 人生のタイムライン

### Phase 1：大学時代 — 模索と挫折

- **大学**: 札幌の4年制大学で心理学を専攻
- **本音の動機**: 心理学への興味以上に「働かなくて良くなるくらいの収入を得る時間が欲しい」という思い
- **詐欺被害**:
  - フリマサイトでの取引をきっかけに詐欺に巻き込まれる
  - 4万円 → 8万円 → 16万円 と次々現金を要求される
  - さらに32万円を要求されたところで親が介入、警察沙汰に
  - ここで得た教訓：「楽して稼ごうとすると痛い目に遭う」
- **副業への挑戦**: 独学でブログを始めるも、大学時代は収益ゼロ

### Phase 2：社会人生活 — 安定と焦り

- **新卒入社**: 地方中小企業（営業職 / 顧客のPC設定など）
- **表向き**: 土日休み・残業ほぼなしの「ホワイト企業」
- **内心の不安**:
  - 「このままではスキルが身につかない」という将来不安
  - 月給のうち車のローン＋家賃で約10万円（給与の約半分）が消える生活
  - 残る金額では自己投資も娯楽も限定的
- **ブログ再挑戦**:
  - 社会人になってからブログで月5,000円を達成
  - しかしGoogle AI（SGE等）の影響でアクセスが激減し挫折

### Phase 3：転機 — スキルプラスとの出会い

- **きっかけ**: YouTubeで三上功太氏（アドネス代表）の動画を視聴
- **行動**: スキルプラスの無料相談を経て11月に契約
- **驚異のスピード感**:
  - 受講開始からわずか **1ヶ月で15,000円のマネタイズに成功**
  - クライアントワークへの恐怖はあったが、講師の伴走サポートで克服
- **覚悟の行動**: スキルアップのために **家と車をすべて売り払う** という決断

### Phase 4：現在 — 受講生から講師へ

- **ヘッドハンティング**: 1ヶ月で成果を出した実績と姿勢が運営に評価される
- **転身**: 入会からわずか1ヶ月後（12月）には講師として教える立場に
- **価値観の変化**:
  - 以前：「お金持ちになりたい」が原動力
  - 現在：教え子の成功や感謝の言葉に幸せを感じるように変化
  - **「他者の人生を豊かにすること」** に情熱を注ぐ

## 4. 強みと差別化ポイント

### ストーリーとしての強み
| 要素 | 内容 | 訴求ポイント |
|------|------|-------------|
| 詐欺被害 | 大学時代にフリマ詐欺で数十万の損失 | 「失敗しても這い上がれる」という共感 |
| 副業の挫折 | ブログ月5,000円→Google AI影響で激減 | 同じ壁にぶつかった人への理解 |
| 経済的プレッシャー | 月給の半分がローン＋家賃 | 「お金がない中でも挑戦できる」説得力 |
| 異常なスピード | 受講1ヶ月で15,000円マネタイズ | 「正しい環境に入れば結果は出る」証明 |
| 覚悟の行動 | 家・車を全て売り払う | 圧倒的行動力 → 本気度の象徴 |
| 講師への転身 | 入会1ヶ月で運営側にスカウト | 才能ではなく「姿勢」で評価された好例 |
| 価値観の変化 | 金→他者貢献 | 受講生に寄り添う講師としての信頼性 |

### 講師としての差別化
- **「0→1」の壁を自ら経験し突破した直近の実体験がある**

- クライアントワークへの恐怖を克服した具体的なプロセスを語れる
- 心理学バックグラウンド × 実体験 → 受講生のメンタル面もフォロー可能

## 5. 動画コンテンツの構成要素（ソース動画の分析）

### 動画の狙い
この動画は「スキルプラスの受講生成功事例」として、以下の層をターゲットにしている：

- 副業を始めたいが何から手をつけていいかわからない社会人
- ブログなどで挫折経験がある人
- 動画編集に興味があるが「自分にできるのか」不安な人
- 高額スクールへの投資を迷っている人

### ストーリーの感情設計
```
[共感] 大学で詐欺被害 → 「自分も騙された経験がある」
    ↓
[共感] 社会人の将来不安 → 「自分もこのまま何も変わらないかも」
    ↓
[共感] ブログ挫折 → 「自分も副業うまくいかなかった」
    ↓
[転換] スキルプラスとの出会い → 「正しい環境に入れば変われる？」
    ↓
[驚き] 1ヶ月で15,000円 → 「え、そんなに早く？」
    ↓
[感動] 家と車を手放す覚悟 → 「この人は本気だ」
    ↓
[希望] 講師に転身 → 「やれば人生変わるんだ」
    ↓
[CTA] 「やらないよりはやったほうがいい」 → 行動への後押し
```

## 9. 動画の特徴と制作上のポイント

### コンテンツとしての強み
- **「0→1」の具体的プロセス**が描かれている（単なる成功報告ではない）
- 詐欺被害という負の経歴を包み隠さず公開 → 信頼感
- 技術習得だけでなく「恐怖の克服」というメンタル面にも言及
- **伴走型サポート**の効果を受講生自身が実証

### 要注意ポイント（客観的な補足）
- アドネス株式会社に対しては2025年12月に特定商取引法に基づく行政処分（是正指示）が出ている
- 高額契約に関するネガティブな口コミも存在する
- 本動画はスキルプラスの公式チャンネルによるプロモーション素材としての性格を持つ
