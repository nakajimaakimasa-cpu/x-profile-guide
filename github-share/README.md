# X プロフィール設計ガイド 〜 AI副業で発信するあなたへ

> スキルプラス受講生向け：Xプロフィールを「売れる導線」に変える完全フロー

---

## このガイドでできること

- Xのプロフィールを **フォローされる構成** に作り変える
- 有料note販売への **自然な導線** を設計する
- 自分の経歴・ストーリーを **最大の武器** に変換する

## 所要時間

約30〜60分（STEP1〜4を順番にやるだけ）

---

## 全体フロー

```
STEP1 自分の素材を棚卸しする（15分）
  ↓
STEP2 発信ジャンルを決める（10分）
  ↓
STEP3 アカウント名を決める（5分）
  ↓
STEP4 自己紹介文を書く（15分）
  ↓
完成！ → すぐXに反映
```

## STEP1: 素材の棚卸し（15分）

### Q3. 挫折・転機のストーリー
- 人生やキャリアで大きな転機は？

## STEP2: 発信ジャンルを決める（10分）

STEP1の回答をもとに、以下の3つの基準でジャンルを絞ります。

### ジャンル選定の3条件

| 条件 | チェック |
|------|----------|
| 語れる実体験・数字がある | ○ or × |
| ターゲットが求めている情報である | ○ or × |
| 毎日発信しても苦にならない | ○ or × |

3つとも○がつくテーマが、あなたの発信ジャンルです。

### ジャンルの方向性（参考例）

自分の強み × トレンド の掛け合わせがベスト：

- 「AI × 副業の始め方」
- 「動画編集 × 案件獲得」
- 「0→1の挑戦ストーリー × ノウハウ発信」
- 「AI自動化 × 収益化」
- 「心理学 × 営業・セールス」

### note販売に繋がるジャンルの条件

- 「知りたい」ではなく **「できるようになりたい」** 欲求に刺さるテーマ
- 無料ポスト（X）で興味を引き、有料note（具体手順）で回収できる構造
- 自分自身の実体験・数字で語れること

## STEP3: アカウント名を決める（5分）

### 公式フォーマット

```
名前｜肩書き or キーワード
```

### 名前の選び方

| タイプ | 例 | 特徴 |
|--------|-----|------|
| 本名（漢字） | 中嶋 | 権威性・信頼感が高い |
| 本名（ひらがな） | あきまさ | 親しみやすい |
| 本名（カタカナ） | ナカジマ | プロ感・スタイリッシュ |
| 本名（英語） | NAKAJIMA | 海外感・ブランド感 |
| ニックネーム | あき | キャッチー・覚えやすい |

### 肩書きの選び方

権威性が高い順：

1. **実績型**: 「受講1ヶ月で講師になった男」
2. **肩書き型**: 「AI副業スクール現役講師」
3. **専門家型**: 「AI副業の専門家」
4. **ストーリー型**: 「3回挫折してAIで逆転した人」
5. **ジャンル型**: 「AI×副業の人」

### 決め方のコツ

- 権威性を出したい → 肩書き型・専門家型
- 親しみやすさ重視 → ストーリー型・ジャンル型
- 迷ったら **「名前｜肩書き型」が最も安定**

### 実例

```
中嶋｜AI副業スクール現役講師
```

## STEP4: 自己紹介文を書く（15分）

### 4段構成フレームワーク

160文字以内で、この4つの要素を順番に入れます。

```
① ストーリー（冒頭）: 挫折 → 転機 → 現在
② 実績（社会的証明）: 数字を含む成果
③ ベネフィット（フォロー理由）: あなたをフォローすると何が得られるか
④ CTA（行動喚起）: noteへの誘導
```

### テンプレート

以下の空欄を埋めるだけで完成します：

```
[挫折・過去] → [転機・きっかけ] → [現在の成果]｜
[スキル・専門性の掛け合わせ]｜
[ターゲットへのメッセージ or 発信内容]｜
[CTA：noteへの誘導文]↓
```

### 記入例

```
副業3回挫折した元営業マン → AI活用で1ヶ月で初収益達成
→ 翌月スクール講師に抜擢｜心理学卒×AI×副業｜
「スキルなし・時間なし」でも始められるAI副業を発信中｜
具体的な手順はnoteにまとめてます↓
```

### 3つのトーン例（同じ素材でも印象が変わる）

**ストーリー重視型:**
> 詐欺で28万失った → ブログ収益0円 → 副業3回挫折 → AIと出会って1ヶ月で初収益 → 翌月に講師へ抜擢｜家と車を売って人生賭けた｜才能ゼロでもAIで逆転できる方法を発信｜0→1の全手順はnoteで公開中↓

**実績×ベネフィット型:**
> 副業3回挫折した元営業マン → AI活用で1ヶ月で初収益達成 → 翌月スクール講師に抜擢｜心理学卒×AI×副業｜「スキルなし・時間なし」でも始められるAI副業を発信中｜具体的な手順はnoteにまとめてます↓

**共感型:**
> 副業で3回負けた。詐欺にも遭った。それでも諦めきれなかった → AIに出会って1ヶ月で人生変わった｜今は講師として同じ悩みの人をサポート中｜「やらないよりはやったほうがいい」｜全部noteに書いてます↓

## 参考：フォロワーが伸びるX発信のコツ

### 投稿ジャンルの黄金比率

- **40%**: ノウハウ・tips（AI副業の具体的な方法）
- **30%**: ストーリー・体験談（挫折→逆転エピソード）
- **20%**: 日常・人間味（趣味、日々の気づき）
- **10%**: 宣伝・CTA（note告知、LINE誘導）

### note販売に繋がるポストの型

**1. 問題提起型**
```
副業を始めて最初にやりがちな失敗3つ：
① ○○
② ○○
③ ○○
全部やった。そして全部AIで解決した。
具体的な方法はnoteに書いた↓
```

**2. 実績公開型**
```
AI自動化を始めて1ヶ月の結果：
・○○
・○○
・○○
やったこと全部noteにまとめました↓
```

**3. ビフォーアフター型**
```
【Before】副業3回挫折、収益0円
【After】AI活用で1ヶ月○○円

何が変わったか？
→ noteで全手順を公開中↓
```
