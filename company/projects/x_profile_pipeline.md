# Xプロフィール作成パイプライン
## 誰でも最高のプロフィールが作れる完全フロー

作成日: 2026-03-16
ステータス: 構築中

---

## 全体フロー概要

```
STEP1 経歴ヒアリング（素材収集）
  ↓
STEP2 分析・戦略設計（ポジショニング決定）
  ↓
STEP3 ロールモデル分析（ベンチマーク）
  ↓
STEP4 プロフィール文案作成（4段構成）
  ↓
STEP5 視覚設計（アイコン・ヘッダー・固定ポスト）
  ↓
STEP6 最終チェック＆公開
```

---

## STEP1: 経歴ヒアリング（素材収集）

### 目的
プロフィールの「原材料」をすべて引き出す。ここで素材の質と量が決まる。

### Q1-1. 基本情報
- 活動名（Xで表示したい名前）は？
- 年代は？（20代/30代/40代…ざっくりでOK）

### Q1-2. 経歴・キャリア
- これまでの職歴・経歴を時系列で教えてください
  - 業界、職種、役職、年数
  - 転職・独立・起業の経験があれば
- 現在の仕事・活動は？

### Q1-3. スキル・専門性
- 自分が「人より詳しい」と思える分野は？（複数OK）
- 資格・受賞歴・メディア掲載などがあれば
- 数字で語れる実績は？（売上、フォロワー数、年数、件数など）

### Q1-4. ストーリー・転機
- 人生やキャリアの中で大きな転機・挫折・成功体験は？
- 「ゼロから○○した」「○○を乗り越えた」系のエピソードがあれば
- なぜ今の活動をしているのか？（原体験・動機）

### Q1-5. 発信の目的
- Xで発信する目的は？（複数選択可）
  - [ ] 認知拡大・ブランディング
  - [ ] 集客・マネタイズ
  - [ ] 仲間・コミュニティづくり
  - [ ] 情報発信・教育
  - [ ] 採用・求人
  - [ ] その他:
- 最終的にXからどこに誘導したい？（LINE、ブログ、サービスページ等）

### Q1-6. ターゲット像
- 誰に届けたい？（年代、職業、悩み、レベル感）
- その人が今抱えている課題・悩みは？
- その人にとってフォローするメリットは何になる？

### Q1-7. 人間味・価値観
- 趣味・好きなこと・日常で大切にしていることは？
- 自分のキャラクターを一言で表すと？（例：熱血、冷静、ユーモア系）
- 失敗談やコンプレックスで、共感を得られそうなものは？

### Q1-8. ロールモデル
- 参考にしたいXアカウントは？（1〜5つ）
- そのアカウントのどこが好き？（雰囲気、文体、構成など）

---

## STEP2: 分析・戦略設計（ポジショニング決定）

### 目的
ヒアリング結果から「何者として発信するか」を決定する。

### 2-1. 専門ジャンル選定（1〜3つに絞る）
STEP1の回答から候補を抽出し、以下の基準で絞り込む：

| 基準 | チェック |
|------|----------|
| 経験・実績がある | 語れるエピソードや数字があるか |
| 需要がある | ターゲットが求めている情報か |
| 継続できる | 毎日発信しても苦にならないか |
| SIMクラスター上有利 | 1〜3つに収まり、一貫性があるか |

### 2-2. ポジショニングマップ
- 選定ジャンル × 独自の切り口 で差別化ポイントを整理
- 「○○ × ○○」の掛け合わせでユニークなポジションを作る

### 2-3. ストーリーライン決定
- 経歴から最もインパクトのあるストーリーを3〜5語に圧縮
- 例：「元公務員→AI起業家」「借金500万→月収300万」「2年間収益0円→復活」

### 2-4. 数字の選定
- プロフィールに入れる実績数字を1〜3個選定
- 基準：具体的、検証可能、インパクトがある

---

## STEP3: ロールモデル分析（ベンチマーク）

### 目的
成功パターンを抽出し、自分のプロフィールに活かす。

### 3-1. 分析項目
各ロールモデルについて以下を分析：

| 分析項目 | 確認ポイント |
|----------|-------------|
| アカウント名 | キーワードの入れ方、文字数 |
| 自己紹介文 前半 | ストーリー＋専門性＋実績の構成 |
| 自己紹介文 後半 | ベネフィット＋人間味の構成 |
| CTA | 行動喚起の内容と配置 |
| アイコン | 顔写真 or イラスト、印象 |
| ヘッダー | キャッチコピー、配色、情報量 |
| 固定ポスト | コンピテンシーポストの型 |

### 3-2. パターン抽出
- 3アカウントの共通点 → 「成功の型」として採用
- 各アカウントの独自点 → 自分に合うものをピックアップ

---

## STEP4: プロフィール文案作成（4段構成）

### 目的
設計書の4段構成フレームワークに沿って文案を作成する。

### 4-1. アカウント名（3案作成）
- 形式：「個人名 | 専門キーワード」or「個人名＠肩書き」
- 全角20文字以内目安
- Grok AIが検索で拾えるキーワードを含める

### 4-2. 自己紹介文（160文字以内）
| 構成 | 内容 |
|------|------|
| ① 冒頭 | ストーリー（3〜5語）＋専門性 |
| ② 実績 | 数字を含む社会的証明 |
| ③ ベネフィット | フォローする理由 |
| ④ CTA | 次のアクション指示 |

- 箇条書き or 区切り文字（｜/→/・）で情報を整理
- SEOキーワードを自然に配置

### 4-3. 複数案の提示
- 最低3パターンを作成
- トーン違い（権威型 / 共感型 / ストーリー型）で比較

---

## STEP5: 視覚設計（アイコン・ヘッダー・固定ポスト）

### 目的
プロフィール文と視覚要素の一貫性を確保する。

### 5-1. アイコン設計
- 400×400px
- 推奨：高画質な顔写真（信頼構築に有効）
- 代替：イラスト（ブランドイメージに合うもの）

### 5-2. ヘッダー設計
- 1500×500px
- 左側2/3にキャッチコピー配置（右側はスマホで隠れる）
- 配色：プロフ全体のトーンと統一
- ツール：Canva推奨

### 5-3. 固定ポスト設計
- 「コンピテンシーポスト」＝自分の専門性を最も証明する投稿
- スレッド形式推奨（滞在時間↑）
- 設計書 第5章の型1（リスト形式）or 型2（ストーリー形式）を使用

---

## STEP6: 最終チェック＆公開

### 公開前チェックリスト
- [ ] アカウント名に専門キーワードが入っているか
- [ ] 自己紹介文の冒頭で「誰向けか」が明示されているか
- [ ] 実績・数字が1つ以上含まれているか
- [ ] フォローのベネフィットが明確か
- [ ] 人間味・失敗談が含まれているか
- [ ] CTA（行動喚起）が設定されているか
- [ ] アイコンが高画質か（400×400px）
- [ ] ヘッダー左側2/3にキャッチコピーがあるか（1500×500px）
- [ ] 固定ポストが「コンピテンシーポスト」か
- [ ] 発信テーマが1〜3つに絞られているか（SIMクラスター）
- [ ] X Premiumに加入したか
- [ ] リポストされる2条件（専門性＋自己呈示欲求）を意識しているか

### 公開後アクション
- 24時間後にプロフィール閲覧数を確認
- 1週間後にフォロー転換率を測定
- 必要に応じてA/Bテスト（名前やCTAを変えて比較）
