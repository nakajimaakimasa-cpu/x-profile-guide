# AIエージェントは「友だち追加」する時代になった。

どうも！みかみです。

ちょっと聞いてほしいことがある。

LINEで「友だち追加」しただけで、AIエージェントが手に入るようになった。

タスク管理。画像生成。スライド作成。ぜんぶやってくれる。

$ しかもそれ、うちが作った。

いや、自分で言うの恥ずかしいんだけどさ。笑

名前は「アディ」。

アドネスのAIエージェントで、LINEで使える超汎用型のやつ。

「なのだ！」って喋る。かわいい。笑

今日はこの「アディ」の話をする。

なぜこれが僕にとって「革命」なのかも、書いていく。

## まず、僕が感じていた「AIの壁」

ここ2年くらい、AIの進化がヤバいって話はもう何百回と聞いてきたと思う。

ChatGPT。Claude。Gemini。

みんな知ってる。

$ でも、「使いこなしてる」人ってどれくらいいる？

正直に答えてほしい。

「ChatGPT、アカウントは作ったけど最近開いてない」
「Claude、名前は聞くけど何が違うか分からない」
「AI使いたいけど、何を聞けばいいか分からない」

心当たりあるでしょ？

$ これ、AIが悪いんじゃないんよ。

「使い方」の問題でもない。

$ 「居場所」の問題なんです。

## AIは「わざわざ会いに行く場所」にいた

考えてみてほしい。

1. ブラウザを開く
2. ChatGPTのサイトにアクセスする

4. 何を聞くか考える
5. プロンプトを打つ

$ ……めんどくさくない？笑

いや、めんどくさいでしょ。

だってこれ、「AIに会いに行ってる」んだもん。

日常生活の中で「よし、今からAI使うぞ」って意気込む。

ブラウザ開いて、URLタイプして。

$ これ、予約制のコンサルタントに会いに行くのと同じ構造なのよ。

でもさ。

$ 本当に使えるパートナーって「会いに行く人」じゃなくない？

隣にいる人でしょ。

いつも横にいて、話しかけたらすぐ答えてくれる。

しかも「なのだ！」って言ってくれる。笑

## だから、LINEにした

LINEのユーザー数、知ってる？

$ 日本で9,700万人。

もはや「使ってない人を探す方が難しい」レベル。

おばあちゃんも使ってる。
小学生も使ってる。
社長も、新卒も、フリーランスも、専業主婦も。

$ みんな毎日、LINEは開く。

ここにAIエージェントが住んでたら、どうなるか。

「わざわざ会いに行く」が、「話しかけるだけ」に変わる。

「何を聞こうか」が、「とりあえずアディに聞いてみるか」に変わる。

$ これが革命なんです。

AIエージェントを「使う」んじゃなくて、「友だち追加する」。

たったそれだけで、あなたの日常にAIエージェントが住み始める。

## 「タスク管理」が、もう別モノになる

例えば、タスク管理。

今まではどうしてた？

Notionにページ作って、カンバン設定して、ラベルつけて、期限設定して……。

$ 管理するための「管理作業」が多すぎたんよね。笑

僕のNotionも例外じゃない。

気づいたら47ページ中39ページが3ヶ月以上放置されてた。

$ タスク管理ツールに管理されてるのは、僕の方だった。

でもアディは違う。

LINEで「今週やること整理して」って送るだけ。

アディが「了解なのだ！」って言って、現状のタスクを聞いてきて、優先順位つけて、今日やるべきことを3つに絞ってくれる。

$ 管理するんじゃない。勝手に整理される。

これがAddnessのAI Goal Breakdownの力。

目標を「目的→ゴール→ToDo」に自動で分解する。

その仕組みが、アディの中に入ってる。

> 「ググるとわかる。アドればできる。」

うちが掲げてるこの言葉。

$ 「アドる」が、LINEの中で起きてる。
