## 世界はゴールで整理される

ちょっと壮大な話から始めさせてください。

人間って、意図を持つ生き物なんですよ。

「お腹すいたな」
「あの仕事片付けたいな」
「人生変えたいな」

大小関係なく、人は常に「こうなりたい」「こうしたい」っていう理想を持ってる。

$この「こうなりたい」が、ゴール。
で、ゴールっていうのは「理想の状態」のこと。
理想があるってことは、今の現実との間に「差分」がある。

この差分を埋めるために、人はゴールを分解していく。

「人生変えたい」
→「まず収入を上げたい」
→「そのためにスキルを身につけたい」
→「今週中にこの講座を申し込む」
→「今日、まずサイトを見る」

ね？
どんどん小さくなっていくでしょ？

で、最終的に「物理的に身体を動かせるサイズ」まで分解されたもの。

$それがタスク。
つまり、タスクっていうのはゴールの最小単位にすぎない。

タスクとゴールは別物じゃない。
同じもの。ただサイズが違うだけ。

そして、その最小ゴール（＝タスク）を実行すると、現実世界に変化が起きる。

サイトを見た。
情報が入った。
判断が変わった。
次のアクションが変わった。
理想自体が更新された。

$ゴール → 分解 → 実行 → 現実が変わる → 感じ取る → ゴール更新。
このループ。
これが、世界の全て。

大げさに聞こえる？

でもマジで、人間の営みって全部これなんよ。

仕事も。恋愛も。健康も。学びも。
全部「理想と現実の差分を、ゴールの分解で埋めていく」。

$世界はゴールで整理される。
で、僕はこの構造を世界で最もうまく扱えるシステムを作った。
