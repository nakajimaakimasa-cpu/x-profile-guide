## 先に結論だけ言っておく

忙しい人のために。

$ 全部使った結論: 「最強のAIエージェント」は存在しない。

あるのは「種族の違い」だけ。

水タイプのポケモンに「火を吹け」って言っても無理でしょ。
AIエージェントも同じ。

$ 4つの種族がある。全部得意分野が違う。

そして、それよりもっと大事なこと。

$ どのエージェントを選ぶかより、「ゴールを設計できるか」の方が100倍重要。

これから全部説明する。

## AIエージェント "4つの種族"

2026年2月現在、AIエージェントは大きく4つの種族に分かれた。

$ 種族1: コーディング特化族（Cursor / Claude Code / Devin）

コードを書く。テストを回す。PRを出す。
「プログラミング」に全振りした種族。
エンジニアにとっては命の恩人。
非エンジニアにとっては「何の話？」って感じ。笑

$ 種族2: 汎用万能族（OpenClaw）

メール処理、スケジュール管理、ファイル整理、スマートホーム操作。
「日常生活の全部」をやろうとする種族。
GitHub 21万6000スター。化け物。

$ 種族3: SaaS内蔵族（Notion / Salesforce / Microsoft）

既存のSaaSの中に「住んでる」AIエージェント。
そのSaaSの中では最強。
でも外には出られない。

$ 種族4: 行動変換族（Addness / アディ）

ゴールを設定して、分解して、「できる」に変換する種族。
「何をすべきか」を考えてくれる唯一の存在。

…はい、自社プロダクトです。笑
でも嘘は言ってないから聞いてほしい。あとで話す。

## 種族1: コーディング特化族

### Cursor — 評価額4.4兆円の化け物エディタ

まずCursor。

$ Cursor: AIが住んでるコードエディタ。月約3,000円。

2026年2月時点で評価額293億ドル。約4.4兆円。
年間売上10億ドル超え。

エンジニアの間では「使ってない奴いるの？」レベルまで来た。

僕が一番使ってるのはAgent Mode。

ファイルを跨いだ編集、ターミナルのコマンド実行、テストの自動修正。
全部AIがやってくれる。

2月のアップデートで「Background Agents」が追加された。

$ これがやばい。

クラウド上の仮想マシンでエージェントが動くから、自分のPCのリソースを食わない。

しかも10〜20タスクを並列で走らせられる。

デスクトップからでも、スマホからでも、Slackからでも指示できる。

$ 月3,000円。バグってる。笑

体験談を1つ。

先週、社内ツールの改修をCursorに任せた。
「このダッシュボードにリアルタイム更新機能つけて」って指示。
Agent Modeが6ファイルを同時に編集して、テスト書いて、全部通して。

$ 20分で完了。人間がやったら丸1日はかかる。

すごいでしょ？

でもね。

Cursorは「コードの書き方」は教えてくれる。

$ でも「何を作るべきか」は教えてくれない。

ここ。超重要。覚えておいて。

### Claude Code — ターミナルの魔術師

次、Claude Code。

$ Claude Code: Anthropicが作った、ターミナルで動くAIコーディングエージェント。

2026年2月にバージョン2.0がリリースされた。

何がヤバいか:

- マルチエージェント: 1つのコマンドで分析・実装・テスト・PR作成を並列実行
- 永続メモリ: プロジェクトの慣習やアーキテクチャパターンを覚えてくれる
- CI/CD統合: GitHub Actions、GitLab CI、Jenkinsにネイティブ対応
- チームモード: 複数人で1つのClaudeを共有

$ 2026年末にはGitHubコミット全体の20%以上がClaude Code経由になるって予測されてる。

20%。5回に1回。やばくない？

使ってみた感想。

$ 上級者には最強。

ターミナルで `claude` って打つだけで始まる。

「このリポジトリのテストカバレッジを80%にして」って言えば、勝手にコード読んで、テスト書いて、走らせて、修正して、PRまで出してくれる。

$ 月2,000〜30,000円（プランによる）。

ただし。

ターミナルに馴染みがない人には敷居が高い。
黒い画面に白い文字がバーッと流れるアレ。
「うっ」ってなる人、正直多いと思う。笑

$ そしてClaude Codeも「何を作るべきか」は教えてくれない。

2回目。

## じゃあ「クエスト」は誰が出すの？

ここからが本題。

$ ゴールの設計。これが2026年、AIエージェント時代の最重要スキル。

「何を作るべきか」
「何を自動化すべきか」
「そもそも今月、何を達成したいのか」

ここを設計できる人は、どのエージェントを使っても成果が出る。
ここが曖昧な人は、月100万払っても何も変わらない。

$ で、ここに「アドる」が入る。

> ググるとわかる。アドればできる。

Googleが「知る」を民主化したように。
Addnessは「できる」を民主化する。

ちょっと聞いてほしい。

ダイエットの方法、知ってるよね？
運動して、食事制限して、早寝早起き。

$ 知ってる。でもできない。笑

英語の勉強法も知ってる。
毎日30分シャドーイング。

$ 知ってる。でもやらない。笑

AIエージェントの「使い方」も知ってる。
Cursorのドキュメント読んだ。Claude Codeの入門記事も読んだ。

$ 知ってる。でも使いこなせてない。

これが「知る→できる」の谷。

情報はもう足りてる。
足りてないのは「知」を「行動」に変換する力。

$ Addnessは、この谷を埋める装置。

「目標を話す → AIが分解 → 今やるべきことが出てくる → やる」

- AI Goal Breakdown: 目標を「目的→ゴール→ToDo」に自動分解
- Focus ToDo: 今この瞬間に集中すべきタスクだけ提示
- Daily Focus: 毎朝「今日やるべきこと」をAIが厳選
- AI Consulting: 24時間対応のAIコンサルタント

$ しかもLINEで使える。

アディ（Addy）っていうキャラクターに友だち追加するだけ。

「今月の売上目標200万なんだけど」って送ったら、やるべきことを分解してくれる。
「来週のプレゼンどう構成すればいい？」って聞いたら、ステップバイステップで出してくれる。

$ 月額? 無料プランあり。

……ね？ 宣伝に聞こえるでしょ？笑

自覚はある。笑

$ でもこれ、本気で全種族の「穴」を埋めるために作ったんよ。

他のエージェントは「HOW」の天才。
Addnessは「WHAT」と「WHY」を設計する。

RPGでいえば「クエストボード」。

$ 両方あって初めて、AIエージェントは本当の力を発揮する。

……って書くと壮大すぎてなんか恥ずかしいな。道徳の教科書の1ページ目みたいなこと言ってる。笑

でもまあ、月8万円払ってたどり着いた結論が「ゴール設計が大事」って、割とリアルでしょ。

## 目的別「これ使え」ガイド

具体的にいこう。

あなたの目的別に、最適なエージェントを整理する。

### コードを書きたい → Cursor

$ 月約3,000円。エンジニアなら問答無用。

Agent Modeで指示すれば、ファイル跨ぎの編集もテスト修正もやってくれる。
Background Agentsで並列作業もできる。
コスパ最強。

### 大規模リファクタリングしたい → Claude Code

$ 月2,000〜30,000円。ターミナルに抵抗がないなら最強。

マルチエージェントで分析→実装→テスト→PR作成まで一気通貫。
永続メモリでプロジェクトの文脈を覚えてくれるのが地味にでかい。
チームで使うならTeam plan（月約6,000円/人）がコスパ最強。

### 日常業務を丸ごと自動化したい → OpenClaw

$ ソフト無料 + API費用（月5,000〜30,000円）。最もカスタマイズ自由。

オープンソースだから好きなように改造できる。
SlackやLINEから指示できる。
ただしAPI費用は使い方次第。ヘビーに使うと月30,000円超えることもある。
自分で設定できる人向け。

### チームのルーティンを効率化したい → Notion Custom Agents

$ Notionクレジット制（$10/1,000クレジット）。5月3日まで無料。

チームでNotion使ってるなら導入しない理由がない。
Q&A自動回答、タスク振り分け、レポート自動生成。
ノーコードで設定できるから非エンジニアでもOK。

### 「そもそも何すればいい？」から整理したい → Addness / アディ

$ 無料プランあり。LINEで友だち追加するだけ。

ゴール設計→分解→今日やることの提示まで全部やってくれる。
「何をすべきか分からない」がスタート地点の人に。
全エージェントの「起点」になる存在。

### 全自律で丸投げしたい → Devin

$ 月約75,000円。予算がある人限定。

「作っておいて」で放置→翌朝完成は確かにすごい。
ただしClaude Codeの5.5倍トークン消費。コスパは要計算。
