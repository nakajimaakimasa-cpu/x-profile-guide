## 文章構造パターン

- **冒頭**: 「どうも！（アドネス株式会社の）みかみです。」+ 状況説明 + フック
- **導入**: 日常的な場面描写 or 衝撃的な事実から入る
- **本文**: 問題提起 → 分析 → 独自の視点 → アドネス哲学への接続
- **締め**: 熱いメッセージ + CTA（DM待ってます）
- **P.S.**: 必ずP.S. / P.P.S. / P.P.P.S. をつける（ここに本音やユーモアを入れる）
- **画像**: 各セクションに写真を入れる前提で「〜の写真」とキャプション的に書く

## ストーリーテリング（ナラティブ）スタイルの特徴

みかみの真骨頂は「人生を語る」時に発揮される。情報記事とは全く異なるモードが存在する。

### ストーリー型の文体特徴
- **「・・・」（沈黙の点列）を多用**: 1〜4行の`・・・`で読者の呼吸を操作する
- **ワンセンテンス・ワンライン**: ストーリーの核心部分は1文ごとに完全改行。「読む」ではなく「めくる」体験
- **内面独白を「」で頻出**: 「なんでだよ」「何してんだろう、俺」のように、その瞬間の生の感情をカギ括弧で直接見せる
- **緩急の心拍操作**: 加速（短文連打）→ 減速（沈黙と内省）を交互に繰り返す
- **ユーモアのリリースバルブ**: 重い話が続いた後に必ず「笑」や脱力系画像キャプションで圧力を抜く
- **反復と変奏**: 同じキーワード（ギブ、女、力など）が記事全体で繰り返されるが、出てくるたびに意味が変わる
- **漫画・ゲーム・映画のメタファー**: 「リアル『龍が如く』」「過ぎ去りし時を求めて」のようなサブカル引用で読者との共通言語を作る
- **手紙構造**: 最後に「過去の自分への手紙」を書き、読者が自分宛と錯覚する二重構造で着地

### ストーリーでよく使うフレーズ・パターン
- 「なんでだよ」（内面の叫び）
- 「何してんだろう、俺」（自省の瞬間）
- 「そう信じて疑わなかった」（信念が崩れる前フリ）
- 「ここで僕は決めた」（決意の宣言）
- 「でも、○○だった」（転換）
- 「そしたら○○」（展開の加速）
- 「完全に○○」（断定的ユーモア: 「完全に詐欺まっしぐらw」）

## 避けたいこと

- 一般的なビジネスメディアっぽい無機質な文章
- 上から目線の説教調（あくまで「一緒にやろう」のスタンス）
- 抽象的すぎて「で？」となる文章（必ず具体例を入れる）
- 情報商材っぽいゴリゴリのセールス感
- 短すぎる記事（基本4000文字以上、ガチ記事は10000文字超え）

## 記事で大切にしたいこと

- 冒頭3行で引き込む。スクロールを止めさせる
- 読者に「考えさせる問い」を投げかける
- 体験談（留置所、東大中退、海外、創業期のダンボール机など）を武器にする
- 必ずアドネス哲学（Well-being、LTR、Give、価値提供）に着地させる
- 最後は「一緒にやろう」で終わる。読者を仲間にするスタンス
- P.S.で人間味を出す（ここが一番正直な部分）
